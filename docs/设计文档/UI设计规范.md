# AI Cultivation UI 设计规范

## 一、设计原则
1. **简洁高效**：减少不必要的视觉元素，突出核心内容，操作路径最短化
2. **一致性**：相同功能和元素在不同页面保持一致的设计和交互
3. **易用性**：符合用户使用习惯，操作反馈清晰，降低学习成本
4. **科技感**：契合AI主题，采用现代简约的设计风格，体现技术感

## 二、色彩系统

## 三、字体系统

### 字体层级
| 层级 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| 大标题 | 24px | 600 | 32px | 页面大标题、重要提示 |
| 一级标题 | 20px | 600 | 28px | 卡片标题、模块标题 |
| 二级标题 | 18px | 500 | 26px | 次级标题、列表项标题 |
| 正文 | 16px | 400 | 24px | 正文内容、普通文本 |

### 字体家族
- 中文：`Inter, -apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", sans-serif`
- 英文/数字：`Inter, SFMono-Regular, Menlo, Monaco, Consolas, "Liberation Mono", "Courier New", monospace`
- 代码：`SFMono-Regular, Menlo, Monaco, Consolas, "Liberation Mono", "Courier New", monospace`

## 四、组件规范

### 按钮
| 类型 | 样式 | 尺寸(高×左右内边距) | 圆角 | 适用场景 |
|------|------|----------------------|------|----------|
| 主按钮 | 背景#165DFF，文字白色 | 48px × 24px<br>40px × 20px<br>32px × 16px | 8px | 主要操作、提交、确认 |
| 次按钮 | 边框#165DFF，文字#165DFF，背景透明 | 48px × 24px<br>40px × 20px<br>32px × 16px | 8px | 次要操作、取消、返回 |
| 文字按钮 | 文字#165DFF，无背景无框 | - | 4px | 链接、不重要操作 |
| 危险按钮 | 背景#F53F3F，文字白色 | 同上 | 8px | 删除、确认危险操作 |

### 卡片
- 圆角：8px
- 阴影：`0 2px 8px 0 rgba(0, 0, 0, 0.08)`
- 内边距：24px（普通）、16px（紧凑）
- 背景：#FFFFFF
- 边框：1px solid #F2F3F5（可选）

### 输入框
- 高度：40px
- 边框：1px solid #C9CDD4，聚焦时边框#165DFF，阴影`0 0 0 2px rgba(22, 93, 255, 0.2)`
- 圆角：8px
- 内边距：左右12px
- 占位符文字颜色：#86909C

### 图标
- 尺寸：24px（大）、20px（中）、16px（小）
- 颜色：默认#4E5969，高亮#165DFF，禁用#C9CDD4
- 风格：线性图标，2px描边，圆角端点

## 六、响应式规范
| 断点 | 宽度范围 | 布局 |
|------|----------|------|
| 移动端 | <768px | 单列布局，侧边栏收起为抽屉，内容占满宽度 |
| 平板 | 768px - 1199px | 两列布局，侧边栏固定宽度240px，内容区自适应 |
| 桌面端 | ≥1200px | 两列布局，侧边栏固定宽度280px，内容区最大宽度1200px居中 |

## 七、交互规范
1. **按钮状态**：
   - 正常：默认样式
   - 悬停：透明度0.9，光标变为指针
   - 点击：透明度0.8，添加轻微缩放效果
   - 禁用：透明度0.4，光标变为not-allowed

2. **反馈机制**：
   - 操作成功：顶部弹出绿色成功提示，3秒后自动消失
   - 操作失败：顶部弹出红色错误提示，需手动关闭或自动消失
   - 加载状态：按钮显示loading动画，页面内容显示骨架屏
   - 表单验证：实时验证，错误提示显示在输入框下方，红色文字

3. **动画效果**：
   - 过渡时间：200ms - 300ms
   - 缓动函数：ease-in-out
   - 适用场景：页面切换、弹窗显示/隐藏、元素状态变化
