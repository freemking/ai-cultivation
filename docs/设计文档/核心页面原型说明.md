# AI Cultivation 核心页面原型说明

## 一、首页

### 页面结构
```
┌─────────────────────────────────────────────────┐
│ 导航栏：Logo、搜索框、通知图标、用户头像        │
├──────────┬──────────────────────────┬───────────┤
│ 侧边栏   │ 内容Feed区               │ 侧边栏    │
│ - 首页   │ - 动态卡片（按时间排序） │ - 推荐小组│
│ - 动态   │   • 用户头像、名称、时间 │ - 热门文章│
│ - 文章   │   • 内容、图片、代码块   │ - 活跃用户│
│ - 小组   │   • 点赞、评论、收藏按钮 │           │
│ - 我的   │ - 分页/无限滚动          │           │
└──────────┴──────────────────────────┴───────────┘
```

### 功能说明
1. **导航栏**
   - Logo：点击返回首页
   - 搜索框：支持全局搜索动态、文章、小组、用户
   - 通知图标：显示未读消息数，点击展开通知列表
   - 用户头像：点击展开个人菜单（个人中心、设置、退出登录）

2. **左侧边栏**
   - 主导航菜单，支持收起/展开
   - 突出显示当前所在页面

3. **内容Feed区**
   - 卡片式展示用户发布的动态，按时间倒序排列
   - 支持无限滚动加载更多内容
   - 每个卡片包含用户信息、发布时间、内容、互动按钮
   - 代码块自动高亮显示，支持复制功能

4. **右侧边栏**
   - 推荐感兴趣的小组
   - 展示热门文章榜单
   - 活跃用户推荐

## 二、发布动态页

### 页面结构
```
┌─────────────────────────────────────────────────┐
│ 导航栏：取消按钮、标题、发布按钮                │
├─────────────────────────────────────────────────┤
│ 内容编辑区                                      │
│ - 文本输入框（支持@用户、#标签）                │
│ - 工具栏：插入图片、插入代码、添加表情          │
│ - 预览区（实时预览发布效果）                    │
├─────────────────────────────────────────────────┤
│ 发布设置区                                      │
│ - 可见范围：公开、仅自己、仅小组可见            │
│ - 添加标签：最多可添加5个标签                   │
│ - 关联小组：选择发布到某个小组                  │
└─────────────────────────────────────────────────┘
```

### 功能说明
1. **编辑器**
   - 支持富文本编辑，自动保存草稿
   - 图片支持拖拽上传，最多可上传9张图片
   - 代码块支持选择编程语言，自动高亮
   - 实时预览功能，切换编辑/预览模式

2. **发布设置**
   - 可见范围灵活控制，保护用户隐私
   - 标签自动联想，支持自定义新标签
   - 关联小组后动态会同时展示在首页和小组内

## 三、小组主页

### 页面结构
```
┌─────────────────────────────────────────────────┐
│ 小组头部：封面图、小组名称、介绍、成员数、加入按钮│
├──────────┬──────────────────────────┬───────────┤
│ 小组导航 │ 小组内容区               │ 小组信息  │
│ - 首页   │ - 小组动态列表           │ - 成员列表│
│ - 讨论   │ - 置顶公告               │ - 小组规则│
│ - 文件   │ - 筛选：全部/精华/仅看管理员          │
│ - 成员   │                          │ - 相关小组│
│ - 设置   │                          │           │
└──────────┴──────────────────────────┴───────────┘
```

### 功能说明
1. **小组头部**
   - 展示小组基本信息，加入/退出按钮醒目
   - 管理员显示"管理"按钮，可进入小组设置

2. **小组导航**
   - 小组内功能模块切换
   - 仅管理员可见"设置"入口

3. **内容区**
   - 展示小组内的所有内容，支持筛选和排序
   - 公告置顶显示，重要信息不被淹没
   - 精华内容标记，方便用户快速找到优质内容

4. **侧边栏**
   - 展示活跃成员、小组规则
   - 推荐相关兴趣小组

## 四、个人主页

### 页面结构
```
┌─────────────────────────────────────────────────┐
│ 个人头部：头像、用户名、简介、关注数、粉丝数、关注按钮 │
├─────────────────────────────────────────────────┤
│ 内容导航栏：动态、文章、收藏、小组              │
├─────────────────────────────────────────────────┤
│ 内容展示区，根据导航切换                        │
│ - 动态：用户发布的所有动态                      │
│ - 文章：用户发布的所有文章                      │
│ - 收藏：用户收藏的内容                          │
│ - 小组：用户加入的所有小组                      │
└─────────────────────────────────────────────────┘
```

### 功能说明
1. **个人头部**
   - 展示用户基本信息和统计数据
   - 自己的主页显示"编辑资料"按钮
   - 他人主页显示"关注/取消关注"、"发私信"按钮

2. **内容导航**
   - 切换展示不同类型的内容
   - 显示各分类的内容数量

3. **内容区**
   - 卡片式展示内容，与首页样式保持一致
   - 支持按时间、热度排序

## 五、文章详情页

### 页面结构
```
┌─────────────────────────────────────────────────┐
│ 导航栏：返回按钮、文章标题、分享按钮            │
├──────────┬──────────────────────────┬───────────┤
│ 目录导航 │ 文章内容区               │ 文章信息  │
│ - 章节1  │ - 文章标题、作者信息、时间│ - 作者信息│
│ - 章节2  │ - 文章内容（Markdown渲染）│ - 相关文章│
│ - ...    │ - 点赞、收藏、分享按钮   │ - 热门标签│
│          │ - 评论区                 │           │
└──────────┴──────────────────────────┴───────────┘
```

### 功能说明
1. **目录导航**
   - 自动提取文章标题生成目录
   - 点击可快速跳转到对应章节
   - 滚动时高亮当前阅读位置

2. **文章内容区**
   - Markdown格式完美渲染，支持数学公式、流程图
   - 代码块支持复制、一键运行（未来功能）
   - 评论区支持回复、点赞，楼中楼展示

3. **侧边栏**
   - 展示作者信息，可快速关注
   - 相关文章推荐，增加内容曝光
   - 热门标签，方便用户探索更多内容

## 六、搜索结果页

### 页面结构
```
┌─────────────────────────────────────────────────┐
│ 搜索栏：搜索关键词、筛选条件                    │
├─────────────────────────────────────────────────┤
│ 结果分类导航：全部、动态、文章、小组、用户      │
├─────────────────────────────────────────────────┤
│ 结果列表区，根据分类切换                        │
│ - 结果卡片，显示匹配度、相关信息                │
│ - 关键词高亮显示                                │
│ - 分页控件                                      │
└─────────────────────────────────────────────────┘
```

### 功能说明
1. **搜索栏**
   - 支持修改关键词重新搜索
   - 高级筛选：按时间范围、热度排序

2. **分类导航**
   - 快速切换不同类型的搜索结果
   - 显示各分类的结果数量

3. **结果列表**
   - 相关度高的内容排在前面
   - 搜索关键词高亮显示，方便用户快速定位
   - 支持分页加载更多结果
